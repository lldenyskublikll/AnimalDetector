## 📌 Про проєкт

**AnimalDetector** — це desktop-застосунок для автоматизованого розпізнавання тварин за допомогою моделей сімейства **YOLO**. Застосунок дозволяє виконувати детекцію та класифікацію тварин на зображеннях, відеофайлах і потоковому відео з камери.

Проєкт був розроблений у межах дипломної роботи на тему:

> **«Система комп’ютерного зору робототехнічної платформи для розпізнавання тварин у динамічних середовищах»**

Основна ідея проєкту полягає у створенні зручного програмного інструменту для запуску навчених нейромережевих моделей, аналізу результатів детекції та збереження вихідних даних у структурованому вигляді.

---

## 🎯 Мета проєкту

Метою розробки є створення програмного забезпечення, яке може використовуватися для:

* тестування моделей комп’ютерного зору на наборах зображень і відео;
* розпізнавання тварин у статичних і динамічних середовищах;
* аналізу результатів роботи нейронної мережі;
* демонстрації можливості використання моделей YOLO у складі програмного забезпечення робототехнічної платформи;
* подальшого розширення під задачі польового моніторингу, екологічних досліджень або автономного спостереження.

## 🧩 Підтримувані режими роботи

### 1. Детекція на зображеннях

Режим дозволяє виконувати аналіз зображень. Після обробки застосунок зберігає анотовані копії файлів і результати детекції.

### 2. Детекція на відео

Режим дозволяє виконувати покадрову обробку відеофайлів. На виході формується відео з нанесеними обмежувальними рамками, назвами класів і значеннями впевненості.

### 3. Детекція на потоковому відео

Режим використовується для роботи з камерою в реальному часі. Він дозволяє перевіряти поведінку моделі у динамічних умовах, коли змінюються освітлення, положення камери, фон і якість відеопотоку.

## 🧠 Моделі

Застосунок орієнтований на використання моделей сімейства **YOLO (v11 та v26)**, навчених для задачі детекції та класифікації тварин.

Модель може виконувати:

* визначення наявності тварини на зображенні або кадрі відео;
* локалізацію об’єкта за допомогою обмежувальної рамки;
* класифікацію виявленого об’єкта за відповідним класом;
* повернення значення впевненості для кожного передбачення.

Для запуску застосунку потрібно виконати команду:

```bash

```

Після цього відкриється графічний інтерфейс, у якому можна вибрати модель, тип вхідних даних, параметри детекції та директорію для збереження результатів.

---

## ⚙️ Основні параметри детекції

| Параметр          | Опис                                                        |
| ----------------- | ----------------------------------------------------------- |
| `confidence`      | Мінімальний поріг впевненості моделі                        |
| `IoU`             | Поріг перетину рамок для фільтрації дубльованих передбачень |
| `image size`      | Розмір зображення, що подається на вхід моделі              |
| `device`          | Пристрій для обчислень: CPU, CUDA або автоматичний вибір    |
| `save labels`     | Збереження координат рамок у TXT-файли                      |
| `save confidence` | Збереження значень впевненості у TXT-файли                  |
| `show labels`     | Відображення назви класів на рамках на зображеннях          |
| `show confidence` | Відображення значень впевненості у рамках на зображеннях    |
